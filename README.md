Install:
npm install --save react-toastify

How to flask admin dashboard:
    before running:
        pip3 install flask
        pip3 install flask-admin
        pip3 install flask_sqlalchemy
    Run inside server:
        python3 -m flask run
    Admin page:
        http://127.0.0.1:5000/admin/
